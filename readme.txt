*Getting Started*

Remember to set ShellLight.Web as startup project

For those who downladed a package you have to unblock all .dll files before the solution builds.

find all assemblies in /lib in Windows Explorer right-click and select Properties.
Click the "Unblock" button at the bottom right of the "General" tab.
